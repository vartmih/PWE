PLAN, WRITE, EXECUTE! API для работы с индивидуальными задачами, с помощью авторизации (личного кабинета). Версия API: {{app_info.pwe}}
API предоставляется работу с CRUD (создать, прочитать, обновить, удалить) операциями для задач. Работа с задачами осуществляется с помощью личного кабинета (регистрация и авторизация). Для аутентификации используется JWT-токен. Токен автоматически передается через cookie. Существует 5 статусов задач:
  • Запланировано
  • В работе
  • Выполнено
  • В ожидании
  • Отменено
Полная документация API доступна по ссылкам: Swagger UI и ReDoc .
